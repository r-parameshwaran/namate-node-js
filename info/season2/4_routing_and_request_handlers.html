Routing and Request handlers

app.use("/") => Matches all the routes

app.use("/hello") => Matches all the routes that has hello

app.use("/test") => Matches all the routes that has test

When we use app.use it handles all type of http methods like get,post,delete

Hello and test has to be exactly same. Also the order matters. If any matching route comes before only it will work. Whenever a request comes it matches from the top

  1. Reading query params: req.query.params
  2. Reading dynamic routes: req.params
