EP 5 - Diving deep into the node js github repo

In node js whenever we create a new module, all the code that we write inside the module is wrapped inside a function then executed when we call require. Variables and functions are private in modules because it is wrapped inside IFFE.

5 step mechanism of require

  1. Resolving the module

    It checks wheather the file is local path, json path or node path

  2. Loading the module

    File context loaded according to the file type

  3. Compiling phase. Wraps inside an IFFE
  4. Code evaluation

    module.export happens and value is returned

  5. Caching the module

We can get __dirname, __filename to get the file path and directory
