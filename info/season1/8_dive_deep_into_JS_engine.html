Dive deep into v8 JS engine

Whenever we pass any code to JS engine it goes through the parsing state.

  1. Lexical analysis (Tokenization)

    The code is broken down into tokens

  2. Syntax analysis

    The syntax analysis of tokens happen and the tokens are converted into AST. (Abstract syntax tree)

Interpreter vs Compiler

Interpreter

  1. runs line by line
  2. Fast initial execution
  3. Interpretter

Compiler

  1. First compilation. High level language code => Machine code
  2. Intially heavy. But executed first.
  3. Compiler

Javascript

Javascript user interpretter also a complier. Javascript uses a mechanism like JIT (Just in Time)
